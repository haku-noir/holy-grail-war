# 聖杯戦争 -Round Table 13- (Web版)

円卓の騎士13人の力を借り、聖杯を奪い合う1対1の対戦カードゲームです。

## 実行方法 (Docker)

Node.jsのバージョン互換性問題を回避するため、Dockerでの実行を推奨します。

```bash
# コンテナのビルドと起動
docker compose up
```
起動後、ブラウザで `http://localhost:5173` にアクセスしてください。

## 実行方法 (ローカル Node.js)

Node.js v20.19+ または v22.12+ が必要です。

```bash
# 依存関係のインストール（初回のみ）
npm install

# 開発サーバーの起動
npm run dev
```

起動後、ターミナルに表示されるURL（例: `http://localhost:5173`）をブラウザで開いてください。

## ゲームルール概要

- **勝利条件**: 5ターン終了時に聖杯を多く持っているプレイヤーの勝利。
- **基本ルール**: お互いにカードを出し合い、数値が大きい方が勝利（例外あり）。
- **聖杯獲得**:
  - **確保（圧勝）**: 数値が大きいカードで勝利。場から聖杯を1つ得る。
  - **強奪（下剋上）**: 数値が小さいカードで勝利。相手から聖杯を1つ奪う。

## CPUの戦略

現在のバージョン（MVP）では、**CPUはランダムに手札からカードを選択**します。
特定の戦略（トリスタンを温存する、相手のカードを予測するなど）は実装されていません。今後のアップデートで思考ルーチンが強化される予定です。
